# 🥖 Panificadora Rosa de Saron

> Site institucional desenvolvido para digitalizar a presença da **Panificadora Rosa de Saron** — uma padaria com mais de 20 anos de tradição no mercado, especializada em pães fresquinhos, bolos caseiros e salgados artesanais.

🔗 **[Acessar o site ao vivo](https://panificadora-rosa-de-saron.vercel.app)**

---

## 💡 Motivação

O projeto nasceu da necessidade de oferecer aos clientes um ponto de contato digital simples, rápido e acessível. O objetivo foi criar uma página que funcionasse bem tanto no celular quanto no computador, sem exigir acesso a redes sociais ou aplicativos específicos.

---

## ✨ Funcionalidades

- 📖 **Cardápio digital** — visualização do cardápio em imagem, abrindo em nova aba
- 📍 **Localização** — link direto para o Google Maps com o endereço da padaria
- 💬 **WhatsApp** — botão de contato direto com o número da padaria
- ⏰ **Horários** — seção clara com os horários de funcionamento da semana e fins de semana

---

## 🎨 Design

O design foi desenvolvido do zero com foco em simplicidade e identidade visual da padaria:

- **Paleta de cores** baseada no tom marrom (`#6b3a13`) e bege (`#f4f1ec`), remetendo ao ambiente aconchegante de uma padaria artesanal
- **Logo circular** com borda e destaque no topo da página
- **Botões arredondados** com efeito de hover suave (`translateY` + escurecimento)
- Layout centralizado com largura máxima de `600px` para boa legibilidade
- Design **mobile-first** — testado e validado em telas de celular

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/marssal-mv/panificadora-rosa-de-saron.git

# Acesse a pasta
cd panificadora-rosa-de-saron

# Abra o arquivo no navegador
# Basta abrir o index.html diretamente, ou usar um servidor local:
npx serve ./
```
