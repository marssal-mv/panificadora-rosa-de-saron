# 🥖 Panificadora Rosa de Saron

Site institucional da **Panificadora Rosa de Saron**, uma padaria com mais de 20 anos de tradição, especializada em pães fresquinhos, bolos caseiros e salgados feitos com carinho.

---

## 📋 Sobre o Projeto

Site simples e responsivo desenvolvido em **HTML e CSS puro**, com foco em facilitar o contato dos clientes com a padaria através de:

- Visualização do **cardápio digital**
- Acesso rápido à **localização no Google Maps**
- Botão direto para conversa no **WhatsApp**
